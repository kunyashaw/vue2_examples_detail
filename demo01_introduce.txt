0、引言
什么是软件框架？
    是一套可以被可被复用的代码编写规范，用来解决或者处理复杂的问题。
框架好处？
    提高代码的复用率、降低模块之间的耦合度、提高开发速度、提高代码质量
    举个例子：
        我要做辆车
            方案1：制动模块（刹车、主动防碰撞。。）、发动机（高科技材质、齿轮、换挡）、车内软装（座椅、方向盘、仪表盘）
            方案2：向各科技公司购买所有技术，拼接下、调教下开卖！
        哪个简单？
            肯定是方案2。

一、VueJS 概述

what?
    Vue.js (读音 /vjuː/，类似于 view) 是由尤雨溪带领团队设计的
        一套渐进式的ui框架（UI框架并不全是像bootStrap一样自带样式类的，
        Vue就是一个没有自带样式类的ui库）;属于在整个前端领域中受欢迎程度前三名的框架
        （注意：不支持ie8以下的浏览器）
    什么是渐进式框架？
        很多其他框架在设计时，只是设计了一个核心库，然后交给用户自己去实现各个模块，
        带来的问题就是体验统一性问题，而渐进式是指step by step,官方已经提供了我们
        开发时需要用到的各个模块，需要用到谁，直接一步一步集成官方的模块即可。

    版本号：V2.4.0

where?
    那些公司在使用VueJS？
        饿了么、滴滴 阿里巴巴(WEEX) GitLab...
    VueJS可以用在哪些地方？
        Vue的目标是通过尽可能简单的 API 实现响应的数据绑定和组合的视图组件。

why?
    方便阅读的中文文档
    容易上手
    速度快
    体积小
    基于组件化的开发方式
    代码的可读性、可维护性比较好

how？
    两种方式：
    ① CLI
        # 全局安装 vue-cli
            npm install --global vue-cli
        # 创建一个基于 webpack 模板的新项目
             vue init webpack my-project
        # 安装依赖，走你
            cd my-project
            npm install
            npm run dev
    ② 直接引入对应的资源文件
